## Global HIV Epidemic Dashboard

### Question
How have new HIV infections and the number of people living with HIV (PLHIV) changed over time across different regions?

### Dataset
Global HIV dataset containing information on new infections, people living with HIV, regions, and yearly trends.

### What I Analyzed
- Trends in new HIV infections over time
- Number of people living with HIV (PLHIV)
- Regional comparisons
- Progress and inequalities across regions

### Key Insights
- Some regions have seen declines in new infections
- Other regions continue to face rising or stagnant infection rates
- There are clear regional inequalities in HIV burden
- Progress varies significantly by region

### Tools Used
Power BI | Power Query
