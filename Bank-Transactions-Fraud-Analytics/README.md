## Project Overview
This project analyzes bank transaction data to monitor account flows, track balances, and identify potentially suspicious activities.

The goal is to support financial institutions in detecting fraud, improving transparency, and understanding customer transaction behavior.

---

## Business Question
How can transaction patterns help identify suspicious activity and improve financial monitoring?

---

## Dataset
The dataset contains simulated bank transaction records including:
- Deposits and withdrawals
- Account balances
- Transaction references
- Customer IDs
- Transaction dates
- Transaction types

---

## What I Analyzed
- Daily and monthly transaction trends
- Inflows vs outflows per account
- Balance changes over time
- Unusual spikes in withdrawals or deposits
- Transaction frequency by account

---

## Key Insights
- Certain accounts showed unusually high withdrawal frequency
- Some transactions occurred in patterns that may indicate risk
- Large sudden balance drops could signal suspicious activity
- Regular transaction behavior was easier to distinguish from anomalies
